# 🎮 俄罗斯方块 - 移动端炫酷版

一款专为移动端优化的俄罗斯方块游戏，支持触摸控制、音效、特效和完整的游戏机制。

✨ 功能特性
 🎯 核心玩法
- ✅ 标准 10x20 游戏网格
- ✅ 7 种经典方块形状（I、O、T、S、Z、J、L）
- ✅ 随机包机制（消除极端随机，体验更公平）
- ✅ 碰撞检测 + 满行消除
- ✅ 等级加速（每消除 10 行，下落速度加快）
- ✅ 分数系统（单行 100 / 双行 300 / 三行 600 / 四行 1000）

🎮 控制方式
- 触摸按钮：底部虚拟按钮（← 左移 / ↻ 旋转 / → 右移 / ▼ 加速下落）
- 长按连续移动：按住按钮持续触发
- 键盘支持**：← → ↑ ↓ 和空格键（电脑端调试可用）

### 🔊 音效系统
- 操作音效（移动、旋转、落地、消除行等）
- 背景音乐（BGM，可静音）
- 移动端首次触摸激活音频（符合浏览器策略）

### ✨ 炫酷特效
- 下落拖尾/残影效果
- 连击 Combo 系统（连续消除触发特效文字和分数加成）
- 消除行粒子特效
- 主题皮肤切换（默认 / 赛博粉 / 复古绿）
- 震动反馈（移动端）

 💾 其他功能
- 最高分记录（LocalStorage）
- 存档/读档（保存当前游戏进度）
- 暂停/继续/结束/重新开始
- 深色/浅色主题切换
- 响应式布局（适配不同屏幕尺寸）

📱 如何开始

方式一：本地服务器（推荐）
bash
# 进入游戏目录
cd /path/to/game

# 启动本地服务器
python3 -m http.server 8888

# 手机/电脑浏览器访问
http://localhost:8888/tetris-mobile-final.html


方式二：直接打开（部分浏览器可能有限制）
- 双击 `tetris-mobile-final.html` 文件
- 或在手机文件管理器中点击并用浏览器打开

方式三：局域网分享
1. 确保手机和电脑在同一 Wi-Fi
2. 查看电脑 IP 地址：`ifconfig | grep "inet "` (Mac) 或 `ipconfig` (Windows)
3. 手机访问 `http://电脑IP:8888/tetris-mobile-final.html`

🎯 Combo 系统说明

连续消除行可以激活 Combo 加成：

| Combo | 显示文字 | 分数加成 |
|-------|---------|---------|
| 1 | "Good!" | 无 |
| 2 | "Great!" | 无 |
| 3 | "Excellent!" | +20% |
| 4-5 | "Combo xN!" | +20% ~ +50% |
| 6-7 | "Super Combo xN!" | +50% |
| 8+ | "Mega Combo xN!" | +100% |

- 连续消除间隔 < 1 秒 → Combo 累加
- 间隔 ≥ 1 秒 或 3 次无消除落地 → 重置 Combo
